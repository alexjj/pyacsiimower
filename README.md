# pyacsiimower
Trim that grass

## Usage

Usage: ./mow.py [options]  
```
  -s, --slow=DELAY    slow delay  
  -f, --fast=DELAY    fast delay  
      --nocolor       self explanatory  
      --miss=RATE     miss per myriad  
      --nowait        do not wait at the end  
  -i, --nointerrupt   Ctrl+C is not your friend anymore
```
